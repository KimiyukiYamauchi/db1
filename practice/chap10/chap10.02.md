```sql

-- 既にデータベースがある場合はいったん削除
drop database if exists `bank`;

-- データベース作成（文字コードはUTF8MB4で日本語対応）
CREATE DATABASE `bank` CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

-- 使用するデータベースを指定
USE `bank`;

```

#### 口座テーブル仕様

| 列名   | 型           | 制約          | 備考                  |
| ---- | ----------- | ----------- | ------------------- |

| 更新日  | DATE        |             |                     |
